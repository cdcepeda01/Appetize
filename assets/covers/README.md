# Appetize local cover assets

Use JPG files for now. Name each image exactly like the item id plus `.jpg`.

Folders:

- `assets/covers/movies/`
- `assets/covers/books/`
- `assets/covers/music/`
- `assets/covers/art/`

Recommended sizes:

- Movies and books: `1200x1800`
- Albums: `1400x1400`
- Artworks: `1600x1200` or a faithful crop that preserves the work
